# Impactly Release Guide

This folder contains the final production-ready APK for the Impactly application.

## 📁 Files
- **Impactly_Final.apk**: The release build of the application.

## 📤 How to Push to GitHub
1. Stage the folder: `git add Impactly_Releases/`
2. Commit: `git commit -m "release: add final APK for testing"`
3. Push: `git push origin main`

## 🔗 Creating a GitHub Release (Recommended)
Instead of just pushing the file to the repo, it is better to:
1. Go to your GitHub repository online.
2. Click on **Releases** -> **Create a new release**.
3. Upload `Impactly_Final.apk` as a binary asset.
4. This allows users to download the APK directly from the "Releases" section.
